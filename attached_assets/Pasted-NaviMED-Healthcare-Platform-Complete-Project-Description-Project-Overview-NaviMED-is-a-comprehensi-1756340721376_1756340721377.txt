NaviMED Healthcare Platform - Complete Project Description
Project Overview
NaviMED is a comprehensive multi-tenant healthcare management platform that connects independent pharmacies with hospitals to streamline prescription management, facilitate insurance claims, manage patient records, and enhance communication across healthcare providers. The platform serves as a centralized hub for healthcare operations with strict data isolation between organizations.

Core Architecture
Technology Stack
Frontend: React with TypeScript, Tailwind CSS for responsive design
Backend: Node.js with Express.js, TypeScript for type safety
Database: PostgreSQL with Drizzle ORM for data management
Authentication: JWT-based authentication with Role-Based Access Control (RBAC)
Deployment: Replit cloud platform with automated workflows
Multi-Tenant Architecture
Enterprise-grade isolation: Complete data separation between organizations
Tenant types: Hospitals, Pharmacies, Laboratories, Platform Administration
Cross-tenant interactions: Controlled data sharing for prescriptions and lab orders
White-label capabilities: Custom branding for each organization
